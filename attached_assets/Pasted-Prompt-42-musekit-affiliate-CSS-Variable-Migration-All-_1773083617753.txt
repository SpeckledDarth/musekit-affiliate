The Fix
1. Replace tailwind.config.ts Entirely
Import the shared preset from @musekit/design-system:

import { tailwindPreset } from "@musekit/design-system";
const config = {
  presets: [tailwindPreset],
  content: ["./src/**/*.{js,ts,jsx,tsx}"],
  // ... keep existing plugins if any
};
This replaces ALL hardcoded hex colors with CSS variable mappings.

2. Fix Analytics Charts
Clicks Area Chart: Replace:

<Area stroke="#4c6ef5" fill="#dbe4ff" ... />
With:

<Area stroke="var(--chart-primary)" fill="var(--primary-100)" ... />
Conversions Bar: Replace:

<Bar fill="#4c6ef5" ... />
With:

<Bar fill="var(--chart-primary)" ... />
Revenue Bar: Replace:

<Bar fill="#e64980" ... />
With:

<Bar fill="var(--chart-secondary)" ... />
Also update radius={[4, 4, 0, 0]} to use var(--chart-bar-radius) if feasible (Recharts may need numeric values — if so, keep as-is since the CSS variable is numeric).
